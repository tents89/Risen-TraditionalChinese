# Risen 1 Traditional Chinese Mod (《異世界》一代繁體中文修正檔)

本專案為遊戲 **[Risen](https://store.steampowered.com/app/40300/Risen/) 《異世界》一代** 的繁體中文在地化模組。 

由於現存的繁體中文補丁僅適用於舊版本，在 Steam 於 2023 年進行大版本更新後皆已失效。為了讓繁中玩家能順利在最新版本的遊戲中體驗劇情，特此製作本修正檔。

---

## 模組特點
* **文本**：文本來源取自 **3DM 2.0版中文化**。
* **修正**：針對繁簡轉換錯誤進行修正。
* **相容最新版本**：適配 **Steam 2023 年更新後** 的遊戲版本（GOG 版本未測試）。

---

## 安裝說明
1. **下載檔案**：下載本專案釋出的最新繁體中文語系包與補丁檔案。
2. **放置路徑**：解壓縮至您的遊戲根目錄中（例如 `Steam\steamapps\common\Risen\`）。
3. **啟動遊戲**：享受繁體中文的遊戲內容。

## 其它
* 若不喜歡全螢幕可以至AppData\Local\Risen的ConfigUser.xml以記事本開啟
並在Window中加入FullScreen="false"來將遊戲視窗化

```
		<Window
			AdapterNumber="0"
			Bottom="1440"
			FOV="60.000000"
			FullScreen="false"  --自己加入
			Left="0"
			RefreshRate="60"
			Right="2560"
			Top="0"
			Vsync="false"
		>
```
